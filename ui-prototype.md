## 一、主窗口 - 首页（计时器状态）

```
┌─────────────────────────────────────────────┐
│  ☐ TimerApp                          ─ □ ✕  │  ← 标题栏（简洁风格）
├─────────────────────────────────────────────┤
│                                             │
│    ┌─────────────────────────────────┐      │
│    │                                 │      │
│    │          23:45                │      │  ← 大号倒计时显示
│    │         ─────────             │      │  ← 进度环/条
│    │                                 │      │
│    │      后执行: 锁屏             │      │
│    │                                 │      │
│    └─────────────────────────────────┘      │
│                                             │
│    [ 暂停 ]  [ 重置 ]  [ 立即执行 ]         │
│                                             │
├─────────────────────────────────────────────┤
│  下一次触发: 15:30  |  生效中 ✓             │  ← 状态栏
└─────────────────────────────────────────────┘
```

**元素说明**:
- 倒计时居中大号显示，字体至少 48px
- 环形进度条直观显示剩余比例
- 暂停/重置/立即执行为主要操作按钮
- 底部状态栏显示下次触发时间和当前是否生效

## 五、提示弹窗（倒计时结束前弹出）- 锁屏场景

```
┌─────────────────────────────────────────┐
│  ⚠️ 即将执行操作                   ─ ✕  │  ← 窗口置顶（模态）
├─────────────────────────────────────────┤
│                                         │
│     ┌─────────────────────────────┐     │
│     │                             │     │
│     │           ⏱️                │     │
│     │                             │     │
│     │    将在 30 秒后锁定屏幕     │     │
│     │                             │     │
│     │    ┌───────────────────┐    │     │
│     │    │ ████████████░░░░░ │    │     │  ← 进度条
│     │    └───────────────────┘    │     │
│     │                             │     │
│     │    剩余延后次数: 2/3        │     │  ← 显示可延后次数
│     │                             │     │
│     └─────────────────────────────┘     │
│                                         │
│     [      立即锁屏      ]              │  ← 主按钮
│                                         │
│     [   延后 5 分钟   ]                 │
│     [   延后 10 分钟  ]                 │  ← 多个延后选项
│     [   延后 30 分钟  ]                 │
│                                         │
│  ─────────────────────────────────────  │  ← 分隔线
│                                         │
│     [     取消本次执行      ]           │  ← 独立区域，避免误触
│                                         │
└─────────────────────────────────────────┘
```

**元素说明**:
- 窗口置顶显示，**模态对话框**（必须先响应才能操作主窗口）
- 大号倒计时提示，醒目但不过度打扰
- 显示剩余可延后次数，避免用户困惑
- 多个延后选项（5/10/30分钟），减少重复操作
- 取消按钮与执行按钮分隔，避免误触

## 十、界面风格规范

### 颜色方案

| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色调 | 深蓝 | `#2B579A` |
| 强调色 | 橙/琥珀 | `#FF9800` |
| 成功 | 绿色 | `#4CAF50` |
| 警告 | 黄色 | `#FFC107` |
| 危险 | 红色 | `#F44336` |
| 背景 | 浅灰 | `#F5F5F5` |
| 文字 | 深灰 | `#333333` |

### 字体规范

| 元素 | 字体 | 大小 | 字重 |
|------|------|------|------|
| 倒计时数字 | Segoe UI | 72px | Light |
| 标题 | Segoe UI | 18px | Semibold |
| 正文 | Segoe UI | 14px | Regular |
| 按钮文字 | Segoe UI | 14px | Semibold |

### 尺寸规范

- 主窗口: 480 x 600 px
- 弹窗: 360 x 280 px
- 按钮高度: 36 px
- 输入框高度: 32 px
- 圆角: 4-8 px
