Windows 定时提醒与系统动作工具

产品说明
按自定义时间间隔倒计时，在到期前提前通知，并执行锁屏或关机等系统动作；支持托盘后台运行与生效规则配置。
主要功能
定时倒计时、提前通知与延后、锁屏/关机、时间段与星期生效规则、循环休息、开机自启、退出密码保护、操作日志。
适用系统
Windows 10 / Windows 11（64 位）
运行要求
关机等功能建议以管理员权限运行；关闭主窗口后程序驻留系统托盘，不会退出。
技术框架

使用说明

以下说明对应当前版本功能，建议首次使用按“快速开始”配置一次。

快速开始

  1. 在主界面选择 15/30/45/60 分钟，或输入自定义分钟数并点击“设置”。
  2. 点击“开始”进入计时，点击“暂停”可临时停止，点击“重置”恢复初始状态。
  3. 点击右上角“设置”选择执行动作（锁屏/关机）并保存。

生效规则

  • 可启用“时间段限制”和“星期限制”，仅在有效规则周期内执行动作。
  • 若当前不在有效周期，主圆环下方会显示红字提示，且自动循环会暂停。
  • 恢复到有效周期后，重新点击“开始”即可继续正常流程。

执行模式

  • 单次执行：完成一次动作后停止，等待手动开始。
  • 循环执行：完成后进入“休息中”，到间隔结束自动进入下一轮。
  • 循环间隔支持 5/10/15 分钟预设和自定义分钟数。

提前通知与动作

  • 可设置“提前通知时间”（不通知/10秒/30秒/1分钟/5分钟）。
  • 通知弹窗支持“立即执行”与“取消”；倒计时结束会自动执行动作。
  • 即使窗口在托盘或最小化状态，提前通知也会弹出。

托盘、日志与安全

  • 点击窗口关闭按钮会进入系统托盘，不会退出程序。
  • “打开日志文件”会直接打开日志；若不存在会自动创建并记录事件。
  • 退出应用需要密码验证；忘记密码可通过密保问题重置。

提示：若你已保存新配置但界面状态未更新，可先“重置”再“开始”重新进入完整流程。
