使用说明

Windows 定时倒计时工具，到期前提醒，并可执行锁屏或关机。

快速使用

  • 选择 15/30/45/60 分钟，或输入自定义时间后点「设置」。
  • 点「开始」计时；「暂停」临时停止，「重置」恢复初始状态。
  • 计时进行中改时间：预设只预选，需点「设置」并确认后才会生效。
  • 点右下角「配置设置」打开系统设置，选择锁屏/关机、循环规则等。

界面与托盘

  • 左下角主题按钮循环切换：深色 → 明亮 → 鲜艳欢快。
  • 关闭主窗口会最小化到托盘，程序继续在后台运行。
  • 到期前会弹出通知，窗口在托盘时同样可见。

其它

  • 可配置时间段、星期生效规则；不在有效周期内不会执行动作。
  • 循环模式完成后进入休息，间隔结束自动开始下一轮。
  • 退出程序需验证密码；忘记密码可通过密保问题重置。
